# PinkSync

PinkSync is a deaf-first video communication platform with professional processing and accessibility features.

## Related Repositories

This project is connected to the following repository:

- **v0-deaf-creator-platform-multi-tenants**: [https://github.com/pinkycollie/v0-deaf-creator-platform-multi-tenants](https://github.com/pinkycollie/v0-deaf-creator-platform-multi-tenants)

### Adding the Remote Repository

To add the v0-deaf-creator-platform-multi-tenants repository as a remote:

```bash
git remote add v0-deaf-creator-platform https://github.com/pinkycollie/v0-deaf-creator-platform-multi-tenants
```

To fetch branches from the remote:

```bash
git fetch v0-deaf-creator-platform
```

## Project Structure

- **app/**: Next.js application pages and layouts
- **components/**: React components including video call and UI components
- **lib/**: Utility libraries
- **public/**: Static assets
- **styles/**: CSS styles

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Build for production:

```bash
pnpm build
```

## Features

- Deaf-first video calling
- Professional video processing
- Accessibility features
- Mux video integration
- Real-time communication with WebSocket support
